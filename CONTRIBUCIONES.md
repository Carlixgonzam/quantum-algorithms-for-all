# Cómo contribuir

Este repo empezó como mis propias notas de estudio, pero está abierto a correcciones y mejoras.

## Si encontrás un error

Abrí un issue señalando el archivo y la línea/fórmula específica. Si tenés la corrección lista, mejor un PR directo.

## Si querés agregar un problema nuevo

Seguí el formato que ya usa `banco-de-problemas/`: enunciado resumido → derivación completa → respuesta en negrita → una línea de "Intuición" al cierre. No agregues solo la respuesta sin la derivación — el valor de este repo está en el razonamiento, no en la opción correcta marcada.

## Estilo

- Notación: $\LaTeX$ inline con `$...$`, bloques con `$$...$$`.
- Español neutro, tono directo — nada de relleno innecesario antes de llegar al punto.
- Cada derivación debe poder verificarse numéricamente en un notebook si el tema lo permite.
