# Banco de problemas

Preguntas tipo examen organizadas por tema, con solución razonada paso a paso — no solo la opción correcta marcada. La idea es poder repasar sin tener que reconstruir la derivación desde cero cada vez.

## Índice

| Archivo | Temas que cubre |
|---|---|
| `costo-de-circuitos-y-compuertas.md` | Size y depth, simulación de circuitos booleanos, eficiencia computacional, conjuntos universales |
| `algoritmo-de-grover.md` | Geometría de Grover, número de iteraciones, caso balanceado, límites del algoritmo |
| `deutsch-jozsa-y-simon.md` | Versión original vs. moderna, caso sin promesa |
| `estimacion-de-fase-y-shor.md` | QFT, mystery gate, orden modular, factorización |
| `fundamentos-y-otros-circuitos.md` | Matrices unitarias, modelo de consultas, swap test |

## Cómo usar este banco

Cada problema sigue el mismo formato: enunciado resumido → derivación → respuesta → una línea de intuición conceptual al final. Esa última línea es la que más me sirve para repasar rápido sin releer toda la cuenta — si puedo reconstruir la intuición de memoria, ya entendí el problema de verdad.
